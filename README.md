# 🎬 IMDB Movie Recommendation System
End-to-end NLP system for movie recommendations using storyline similarity, TF-IDF vectorization, Cosine Similarity, and Streamlit dashboard — built on real scraped IMDB 2024 data.

---

### Tech Stack
* Python
* Selenium
* Pandas & NumPy
* NLTK
* Scikit-learn
* Streamlit
* Matplotlib & Seaborn

---

### Features
* IMDB data scraping
* Data cleaning & EDA
* NLP text preprocessing
* TF-IDF vectorization
* Cosine similarity engine
* Interactive dashboard

##  App Preview

> Run `streamlit run imdb_app.py` to launch the app locally.
