# IMDB-Movie-Recommendation-System
Scraped 4305 IMDB 2024 movies using Selenium | Built End-to-end NLP system for movie recommendation  using TF-IDF + Cosine Similarity and Interactive Streamlit dashboard

---

### Tech Stack
* Python
* Selenium
* Pandas & NumPy
* NLTK
* Scikit-learn
* Streamlit
* Matplotlib & Seaborn

---

### Features
* IMDB data scraping
* Data cleaning & EDA
* NLP text preprocessing
* TF-IDF vectorization
* Cosine similarity engine
* Interactive dashboard

##  App Preview

> Run `streamlit run imdb_app.py` to launch the app locally.
